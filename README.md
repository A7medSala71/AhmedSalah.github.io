# Ahmed Salah - Portfolio

A modern, responsive portfolio website showcasing expertise in Machine Learning, AI, Full-Stack Development, and Embedded Systems. Built with clean HTML, CSS, and vanilla JavaScript, the portfolio features a professional design with smooth animations and optimal performance.

## Features

- **Modern Design**: Professional dark theme with cyan accents and glassmorphism effects
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices
- **Navigation System**: Sticky navigation bar with smooth scrolling across all pages
- **Featured Projects**: Showcase of key projects with technology tags
- **Skills Section**: Organized technical skills by category with visual hierarchy
- **Certifications**: Display of professional certifications and achievements
- **Contact Information**: Multiple channels for reaching out (email, LinkedIn, GitHub)
- **Smooth Animations**: Fade-in effects, hover transitions, and interactive elements
- **SEO Optimized**: Meta tags and semantic HTML structure

## Installation & Usage

### Local Setup
1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through pages using the navigation menu

### No Build Process Required
This is a static website with no dependencies, build tools, or server required.

### Customization

**Update Personal Information**:
- Edit HTML files to change name, email, phone number
- Update content in `data/content.json`
- Replace resume PDF in `assets/` folder

**Modify Styles**:
- Edit `styles/style.css` to change colors, fonts, or layout
- Use CSS variables for consistent theming
- Responsive breakpoints are at 768px and 480px

**Add/Remove Projects**:
- Edit `projects.html` to add or remove project cards
- Update project details and technology tags

## Design System

### Color Palette
- **Primary Background**: #0f172a (Deep Blue)
- **Secondary Background**: #1e2942
- **Accent Color**: #22d3ee (Cyan)
- **Text Primary**: #f1f5f9 (Off-white)
- **Text Secondary**: #cbd5e1 (Light Gray)
- **Text Tertiary**: #94a3b8 (Medium Gray)

### Typography
- **Font Family**: System fonts (-apple-system, BlinkMacSystemFont, Segoe UI, Roboto)
- **Heading Size**: 2.5rem - 3.5rem
- **Body Text**: 0.95rem - 1rem
- **Line Height**: 1.6

### Components
- **Cards**: Gradient backgrounds with border and backdrop blur effect
- **Buttons**: Primary (gradient background) and Secondary (outlined)
- **Tags**: Rounded badges with subtle backgrounds
- **Links**: Underline on hover with color transition

## Customization Guide

### Changing Colors
Edit the color values in `styles/style.css`:
- Primary accent: Replace `#22d3ee` throughout the file
- Background colors: Modify `#0f172a` values
- Text colors: Update `#f1f5f9`, `#cbd5e1`, etc.

### Adding New Projects
1. Open `projects.html`
2. Duplicate a project card block
3. Update project title, description, and tags

### Updating Resume
1. Replace `assets/Ali_Kareem_resume.pdf` with your resume
2. Update the link in index.html and other pages if needed

### Social Links
Update the following in `contact.html`:
- LinkedIn URL
- GitHub URL
- Email address
- Phone number
